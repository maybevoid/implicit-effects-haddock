data NoEff #

The trivial effect NoEff have singleton operation and can be trivially satisfied for all Effect eff. It is the identity for the Control.Effect.Implicit.Base.Union type operator such that:

forall ops . ops ∪ NoEff ~= NoEff ∪ ops ~= ops

which by ops1 ~= ops2 means ops1 ⊇ ops2 and ops2 ⊇ ops1, i.e. they satisfies equivalent set of OpsConstraints up to casting equivalents by OpsCast.
